## Учебный сервис "YamDB" 



## Deploy
В корневой папке 
- Создать файл `.env` по примеру файла `.env.sample`
- Запустить `docker-compose up -d`

## Технологии
- API на "Django + DRF"
- Тестирование на "Pytest"
- БД PostgreSQL
- Proxy Nginx
- Контейнеризация с помощью Docker
